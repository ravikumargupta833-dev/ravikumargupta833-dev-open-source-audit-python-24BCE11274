## 📌 About This Project

This project is part of the Open Source Software course. The main goal was to understand how open-source tools actually work, not just theoretically but also practically using Linux.

I chose Git because it is widely used in software development and has a strong open-source background.

---

## ⚙️ Scripts Included

### 1. System Identity Report
This script displays basic system details like kernel version, user, uptime, and OS information. It works like a welcome screen for the system.

---

### 2. FOSS Package Inspector
This script checks whether Git is installed or not. It also shows the version and gives a short description of the software.

---

### 3. Disk and Permission Auditor
This script goes through important system directories and shows their size and permissions. It also checks the Git configuration file.

---

### 4. Log File Analyzer
This script reads a log file and counts how many times a keyword (like "error") appears. It also shows the last few matching lines.

---

### 5. Open Source Manifesto Generator
This is a small interactive script. It asks a few questions and generates a personal open-source manifesto saved in a text file.
